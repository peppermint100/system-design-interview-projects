# 가상 면접 사례로 배우는 대규모 시스템 설계 구현 프로젝트

## 소개
이 리포지토리는 가상 면접 사례로 배우는 대규모 시스템 설계라는 책을 읽고 관심이 가는 도메인의 설계를 따라 최대한 비슷하게 구현한 리포지토리입니다. 
구현 상황에 맞게 더미로 작성한 부분도 있고, 책에서 조금 애매하다 싶은 부분을 조금 더 개선하기도 합니다.
